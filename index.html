BSB Lookup - Australia BSB Number Checker

BSB Numbers play a crucial role as concise identifiers for bank details, providing a convenient alternative to remembering lengthy strings of text. Instead of burdening customers with extensive information, a brief numerical code is assigned, facilitating easy interpretation by banks and businesses to access crucial bank account details.

Breaking down the BSB number, the first two digits represent the parent financial institution, the third designates the state of the branch, and the fourth to sixth digits specify the branch itself. For instance, analyzing the BSB 033088 reveals the following breakdown:

  • 03: Westpac Banking Corporation
  • 3: Victoria
  • 088: 383 Chapel Street, Prahran

Although this numbering convention is typically followed by banks, building societies, and credit unions might deviate. However, this divergence doesn't impact the ability to decipher bank details. Some financial institutions might opt for a centralized BSB for all accounts, exemplified by Suncorp's use of 484799 for all deposit accounts, irrespective of the branch or state of account origination.

In terms of the third digit in the BSB, indicative of the state of origin, the following guide can be referenced:

  • 2: Australian Capital Territory, New South Wales
  • 3: Victoria
  • 4: Queensland
  • 5: South Australia, Northern Territory
  • 6: Western Australia
  • 7: Tasmania
What is the BSB number in banking?

The BSB number (Bank/State/Branch number) is a numerical code used in banking to identify a specific branch and location of a customer's account. It facilitates the efficient processing of transactions and is primarily used within Australia.

What are the first 3 digits of the BSB?

The first 3 digits of the BSB number represent the following:

  • The first two digits specify the parent financial institution.
  • The third digit specifies the state where the branch is located.

Is BSB a Swift number?

No, the BSB (Bank/State/Branch) number is not the same as a SWIFT (Society for Worldwide Interbank Financial Telecommunication) number. BSB is used specifically in the Australian banking system to identify branches within the country. On the other hand, SWIFT codes are used internationally for identifying banks and financial institutions globally.
